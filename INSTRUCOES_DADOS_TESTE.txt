=== INSTRUÇÕES PARA INSERIR DADOS DE TESTE ===

Este arquivo contém as instruções para popular o banco de dados com dados simulados para testar todas as funcionalidades do Portal de Notícias.

1. ARQUIVO DE DADOS DE TESTE:
   - O arquivo 'dados_teste_completos.sql' contém todos os comandos INSERT necessários
   - Este arquivo popula todas as tabelas do sistema com dados realistas

2. COMO EXECUTAR OS DADOS DE TESTE:

   OPÇÃO A - Via linha de comando MySQL:
   mysql -u root -p portal_noticias < dados_teste_completos.sql

   OPÇÃO B - Via phpMyAdmin ou outro cliente MySQL:
   1. Abra o phpMyAdmin ou seu cliente MySQL preferido
   2. Selecione o banco de dados 'portal_noticias'
   3. Vá para a aba 'SQL' ou 'Importar'
   4. Cole o conteúdo do arquivo 'dados_teste_completos.sql'
   5. Execute os comandos

   OPÇÃO C - Via MySQL Workbench:
   1. Abra o MySQL Workbench
   2. Conecte-se ao servidor MySQL
   3. Abra o arquivo 'dados_teste_completos.sql'
   4. Execute o script completo
